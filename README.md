# Urban Guardian 🛡️

**App móvil de seguridad personal para trayectos nocturnos**

Diseñada para combatir la "fatiga de seguridad" en trabajadores por turnos, repartidores (riders) y estudiantes. Prioriza la prevención sobre la reacción.

## 🚀 Demo en vivo

👉 **[Ver aplicación en GitHub Pages](https://TU-USUARIO.github.io/urban-guardian)**

## ✨ Características

- **Mapa realista** con calles, edificios y GPS simulado
- **Ruta segura iluminada** con animación SVG desde tu ubicación al destino
- **SOS discreto** con long-press y PIN falso de cancelación
- **Red de guardianes** - voluntarios y guardias activos cercanos
- **Gamificación** - niveles, XP y logros por reportar incidentes
- **Refugios cercanos** - farmacias 24h, gasolineras, comisarías

## 📱 Vistas

| Vista | Descripción |
|-------|-------------|
| **Mapa** | Navegación con rutas seguras, heatmap de seguridad |
| **Reportar** | Micro-incidentes (calle oscura, grupo sospechoso, etc.) |
| **Red** | Contactos de confianza + Guardianes locales (P2P) |
| **Perfil** | Dashboard de seguridad, estadísticas, ajustes SOS |

## 🛠️ Tecnologías

- HTML5 + CSS3 puro (Flexbox/Grid)
- JavaScript vanilla (SPA)
- SVG para rutas animadas
- FontAwesome + Google Fonts Inter
- GitHub Pages para hosting
